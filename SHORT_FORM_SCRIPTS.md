# Short-Form Scripts: 7 x 60-Second Clips
## For YouTube Shorts, TikTok, and Instagram Reels

### Why 7 Clips?
- **Maximum platform reach:** Each clip targets different timing/audience moment
- **Compound leverage:** Each clip drives traffic back to long-form video
- **Testing:** See which hooks resonate most
- **Content variety:** 2-3 weeks of daily content from one video

### Publishing Strategy
- **Post 3 to TikTok** (space 4-6 hours apart)
- **Post 3 to YouTube Shorts** (daily or every other day)
- **Post 2 to Instagram Reels** (once daily)
- **Include link to full video** in bio, description, or comments (every variation)

### Format for Each Clip
- **Length:** 45-60 seconds (TikTok loves under 60)
- **Opening Hook:** 3-5 seconds (stop scroll)
- **Content:** 40-50 seconds (deliver promise)
- **CTA:** 2-3 seconds (send to main video)
- **Font size:** 18pt minimum (readable on phone)
- **Text overlays:** Key phrases in white or bright yellow

## Universal Template for All Clips

Every short-form should follow this structure:

```
HOOK (3-5 sec)
- Show problem visually
- Voiceover: short, punchy

PATTERN BREAK (3-5 sec)
- Show solution visually
- Voiceover: reframe the problem

BODY (40-50 sec)
- Explain in detail
- Show examples/code
- Build understanding

TEXT OVERLAYS
- Key phrases in white/yellow
- Appear at 0:10, 0:25, 0:40
- Make sense without sound

CTA (2-3 sec)
- Direct to full video
- Show link/QR code
- Voiceover: clear, simple
```

## Export Specifications

### Video Format
- Resolution: 1080x1920 (vertical)
- Frame rate: 30fps (TikTok/Instagram standard)
- Codec: H.264
- Bitrate: 5-8 Mbps

### Audio Format
- Sample rate: 48kHz
- Channels: Stereo
- Format: AAC
- Bitrate: 128-256 kbps
